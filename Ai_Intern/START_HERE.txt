TROUBLESHOOTING:
================================================================================

Problem: "No module named 'X'"
Solution: pip install -r requirements.txt

Problem: "No PDF files found"
Solution: Make sure PDF files are in the "data" folder

Problem: "Vector database not found"
Solution: Select option 1 first to process documents

Problem: Processing is slow
Solution: This is normal! First time takes a few minutes.
